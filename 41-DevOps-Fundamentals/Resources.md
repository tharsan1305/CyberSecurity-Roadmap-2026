# Day 41 - Resources

## Key Terms Glossary
- **CI**: Continuous Integration
- **CD**: Continuous Delivery / Continuous Deployment
- **Pipeline**: the automated sequence of build/test/deploy steps
- **DevOps Loop**: Plan, Code, Build, Test, Release, Deploy, Operate, Monitor (cyclical)
