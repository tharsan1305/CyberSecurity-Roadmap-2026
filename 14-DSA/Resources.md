## Key Concepts Reference
| Structure | Access | Insert | Use Case |
|---|---|---|---|
| Array | O(1) | O(n) | Fixed, indexed data |
| Linked List | O(n) | O(1) at head | Frequent insertions |
| Stack (LIFO) | O(1) top | O(1) | Undo, function calls, bracket matching |
| Queue (FIFO) | O(1) front | O(1) | Task scheduling, BFS |
| Tree | O(log n) avg | O(log n) avg | Hierarchical data |
| Graph | varies | varies | Networks, relationships, attack paths |

## Big O Quick Reference
| Notation | Name | Example |
|---|---|---|
| O(1) | Constant | Array index access |
| O(log n) | Logarithmic | Binary search |
| O(n) | Linear | Single loop |
| O(n log n) | Linearithmic | Merge sort, built-in Python sort |
| O(n²) | Quadratic | Bubble sort, nested loops |
