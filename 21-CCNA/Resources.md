## Commands Reference
| Purpose | Command |
|---|---|
| Set hostname | `hostname <name>` |
| Set privileged exec password | `enable secret <password>` |
| Encrypt all passwords | `service password-encryption` |
| Save config | `copy running-config startup-config` |
| View interface summary | `show ip interface brief` |
| View routing table | `show ip route` |
| View OSPF neighbors | `show ip ospf neighbor` |
| View VLAN assignments | `show vlan brief` |

## Key Terms Glossary
- **Running-config**: currently active configuration in RAM (lost on reboot if not saved)
- **Startup-config**: saved configuration loaded on boot
- **MPLS**: Multiprotocol Label Switching, WAN forwarding technology
- **Neighbor Adjacency**: relationship formed between routers running the same dynamic routing protocol
