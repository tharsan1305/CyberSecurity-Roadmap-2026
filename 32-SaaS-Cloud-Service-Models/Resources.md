## Key Terms Glossary
- **IaaS**: Infrastructure as a Service
- **PaaS**: Platform as a Service
- **SaaS**: Software as a Service
- **Multi-tenant**: shared infrastructure serving multiple customers (typical of public cloud)
- **Shared Responsibility Model**: security division between cloud provider and customer

## Further Reading
- AWS Shared Responsibility Model official page (aws.amazon.com/compliance/shared-responsibility-model)
- NIST Definition of Cloud Computing (SP 800-145) - the formal standard defining IaaS/PaaS/SaaS
