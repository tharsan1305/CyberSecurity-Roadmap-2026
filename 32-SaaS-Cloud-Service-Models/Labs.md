## Objective
Since this topic is conceptual, today's lab is a mapping/research exercise to solidify understanding before hands-on AWS labs begin (Topic 33 onward).

## Task 1: Classify Real Tools
For each tool/service below, classify it as IaaS, PaaS, or SaaS, and justify briefly:
- AWS EC2
- Heroku
- Gmail
- AWS Lambda
- Microsoft 365
- DigitalOcean Droplets
- AWS Elastic Beanstalk
- Salesforce

## Task 2: Shared Responsibility Breakdown
For an AWS EC2 instance (IaaS), list out specifically:
- What AWS is responsible for securing
- What YOU would be responsible for securing

Do the same for an AWS S3 bucket (closer to a managed storage service) and compare how the split differs.
