# Day 17 - Resources

## Commands Reference
| Purpose | Windows | Linux |
|---|---|---|
| Show IP config | `ipconfig /all` | `ip addr` |
| Active connections | `netstat -ano` | `ss -tulnp` |
| Trace route | `tracert <host>` | `traceroute <host>` |

## OSI Model Quick Reference
| Layer | Number | Examples |
|---|---|---|
| Application | 7 | HTTP, FTP, DNS |
| Presentation | 6 | SSL/TLS, encoding |
| Session | 5 | Session management |
| Transport | 4 | TCP, UDP |
| Network | 3 | IP, routers |
| Data Link | 2 | MAC addresses, switches |
| Physical | 1 | Cables, signals |
