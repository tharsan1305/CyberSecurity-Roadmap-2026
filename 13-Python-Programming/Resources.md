## Commands Reference
| Purpose | Command/Syntax |
|---|---|
| Run a script | `python3 script.py` |
| Install a package | `pip install requests --break-system-packages` |
| Open file for writing | `open("file", "w")` |
| Open file for reading | `open("file", "r")` |
| Open file for appending | `open("file", "a")` |
| Regex search | `re.search(pattern, string)` |

## Key Terms Glossary
- **Mutable vs Immutable**: whether an object can be changed after creation (list=mutable, tuple=immutable)
- **Class/Object**: a blueprint (class) and its instance (object) in OOP
- **Inheritance**: a class deriving behavior from a parent class
- **Module**: a reusable file of Python code (`import module`)
