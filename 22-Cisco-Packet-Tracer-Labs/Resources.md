## Troubleshooting Command Reference
| Purpose | Command |
|---|---|
| Interface status summary | `show ip interface brief` |
| Full running config | `show running-config` |
| VLAN assignments | `show vlan brief` |
| Routing table | `show ip route` |
| OSPF neighbors | `show ip ospf neighbor` |
| Test connectivity | `ping <ip>` |
| Trace path | `tracert <ip>` |

## Key Terms Glossary
- **IP Addressing Table**: a planning document mapping subnets/devices to IP ranges before configuration
- **Fault Injection**: deliberately introducing errors to practice diagnosis (common training technique)
