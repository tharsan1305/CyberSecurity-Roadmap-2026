## Key Terms Glossary
- **Lateral Movement**: an attacker moving from one compromised system to others within a network
- **Micro-segmentation**: dividing a network into small isolated zones with independent access controls
- **Conditional Access**: access policies that evaluate context (identity, device, location) before granting access
- **Assume Breach**: a security design philosophy that assumes attackers may already be inside the network
- **Device Posture/Compliance**: a check on whether a device meets security requirements (patched, encrypted, etc.) before granting access
