## Commands Reference
| Purpose | Command |
|---|---|
| Clone a fork | `git clone https://github.com/<user>/<repo>.git` |
| Add upstream remote | `git remote add upstream <original-repo-url>` |
| Fetch upstream changes | `git fetch upstream` |
| Merge upstream into fork | `git merge upstream/main` |
| Push new branch | `git push origin <branch-name>` |

## Key Terms Glossary
- **Fork**: a personal copy of someone else's repository
- **Pull Request (PR)**: a request to merge changes with review
- **Upstream**: the original repository a fork was created from
- **GitHub Actions**: GitHub's built-in CI/CD automation platform
- **Kanban Board**: visual task tracking (To Do / In Progress / Done columns)
