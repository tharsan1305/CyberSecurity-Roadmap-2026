## Commands Reference
| Purpose | Command |
|---|---|
| Confirm identity | `aws sts get-caller-identity` |

## Key Terms Glossary
- **Region**: geographic AWS location
- **AZ**: Availability Zone, isolated data center within a region
- **Free Tier**: limited free usage program (Always Free / 12 Months Free / Trials)
