# Day 11 - Resources

## Commands Reference
| Purpose | Command |
|---|---|
| Initialize repo | `git init` |
| Stage changes | `git add .` |
| Commit | `git commit -m "message"` |
| Push | `git push origin main` |
| Pull | `git pull origin main` |
| Create branch | `git branch <name>` |
| Switch branch | `git checkout <name>` / `git switch <name>` |
| Create + switch | `git checkout -b <name>` |
| Merge branch | `git merge <branch>` |
| View history | `git log --oneline --graph` |
| Check status | `git status` |

## Key Terms Glossary
- **Branch**: an independent line of development
- **Merge Conflict**: occurs when Git cannot auto-resolve differing changes to the same file content
- **Fast-Forward Merge**: merge with no divergent history, pointer simply advances
- **HEAD**: pointer to your current branch/commit
